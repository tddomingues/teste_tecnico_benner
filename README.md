# Teste Técnico - Programador C#

Esse projeto tem o objeto de fazer um CRUD de pedidos, pessoas (clientes) e produtos.

### Tecnologias e Ferramentas
* C#
* .NET Framework 4.6
* WPF
* Dados em JSON 
* Biblioteca Newtonsoft.Json

# Como instalar o projeto?

1. Ter o Visual Studio.
2. Faça o download ou clone este repositório para sua máquina.
3. Abra o arquivo `.sln` no Visual Studio.
4. O Visual Studio deve restaurar automaticamente o pacote `Newtonsoft.Json`. Caso não ocorra, vá em "Gerenciador de Pacotes do NuGet" e clique em Restaurar.
5. Clique no botão "Iniciar" para rodar a aplicação.
