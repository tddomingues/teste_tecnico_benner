# Teste Técnico - Programador C#

Esse projeto tem o objeto de fazer um CRUD de pedidos, pessoas (clientes) e produtos.

### Tecnologias e Ferramentas
* C#, 
* .NET Framework 4.6 (como pedi o teste), 
* WPF, 
* Dados em JSON 
* Biblioteca Newtonsoft.Json

#Como instalar o projeto?

1. Ter o Visual Studio instalado com a carga de trabalho de "Desenvolvimento para desktop com .NET".
2. Faça o download ou clone este repositório para sua máquina.
3. Abra o arquivo `.sln` no Visual Studio.
4. O Visual Studio deve restaurar automaticamente o pacote `Newtonsoft.Json`. Caso não ocorra, vá em "Gerenciador de Pacotes do NuGet" e clique em Restaurar.
5. É fundamental que o tipo de tela esteja selecionada no App.xaml para:
a) StartupUri="Views/CadastrarPedido.xaml"> ou StartupUri="Views/CadastrarProduto.xaml"> ou StartupUri="Views/CadastrarPessoa.xaml">
6. Clique no botão "Iniciar" para rodar a aplicação.
